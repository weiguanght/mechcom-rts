MechCom Web - 网页版即时战略游戏

这是一个基于 HTML5 Canvas 和原生 JavaScript 开发的 2D 即时战略 (RTS) 游戏原型。灵感来源于经典 RTS 游戏《MechCom 3》。

🎮 游戏特色

三大阵营:

Quadrumm (蓝方): 平衡型，拥有双足机甲与闪电武器。

Trionic (绿方): 科技型，拥有高速H型轮式机甲与等离子武器。

Sphenix (红方): 资源型，拥有重装H型履带机甲与激光武器。

自定义组装: 在“单位设计”面板中自由组合武器与底盘，创造独特的战斗单位。

区域占领机制: 清除区域内的敌人并驻守以占领资源点。

完整科技树: 从采矿场到高级实验室，体验完整的 RTS 建造流程。

🕹️ 操作指南

鼠标左键: 选择单位 / 点击按钮。

鼠标双击: 选中屏幕内所有同类型单位。

鼠标右键: 移动单位 / 攻击目标。

WASD / 方向键: 移动地图视野。

点击小地图: 快速跳转视野。

🚀 如何运行

本项目为单文件应用。

下载 index.html。

直接用浏览器打开即可游玩。

🛠️ 技术栈

HTML5 Canvas

Vanilla JavaScript (无第三方库依赖)

CSS3 Flexbox (用于 UI 布局)

此项目为原型演示，仅供学习交流。
